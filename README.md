# Лабораторная работа 5: Разработка SPA на React

## Цель работы
Разработка одностраничного приложения (SPA) на React с подключением к бэкенд-сервису и реализацией функционала фильтрации.

## Технический стек
- **Фронтенд**: React + TypeScript + React-Bootstrap
- **Бэкенд**: REST API + База данных
- **Хранение файлов**: MinIO
- **Сборка**: Vite
- **HTTP-запросы**: Fetch API

## Бэкенд структура
- REST API endpoints для услуг
- Модели данных для услуг
- Интеграция с MinIO для хранения изображений
- База данных для персистентного хранения

## Функциональные требования

### Страницы приложения

#### Главная страница
- Статическое описание сервиса
- Welcome-контент

#### Страница списка услуг
- Отображение карточек услуг
- Система фильтрации:
  - По диапазону дат
  - По названию (поиск)
