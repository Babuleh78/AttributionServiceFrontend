# Лабораторная работа 8

## Цели работы

- **Межсервисное взаимодействие** - организация коммуникации между основным и асинхронным сервисами
- **Реализация асинхронности** -  внедрение отложенных операций с задержкой выполнения
- **Внедрение Real-time web** - обеспечение актуального отображения данных через short polling
- **Создание интерфейса модератора**  разработка административного функционала для управления заявками

## Основные задачи

Разработка асинхронного сервиса на альтернативном языке программирования (Golang)
Реализация HTTP-метода для выполнения отложенных действий с задержкой 5-10 секунд
Интеграция механизма псевдо-авторизации через токен-ключ
Создание интерфейса модератора с функциями фильтрации и управления статусами
Внедрение short polling для актуального отображения статусов заявок
Обновление диаграмм последовательности и развертывания системы

## Технологии

gRPC - для межсервисного взаимодействия
Асинхронные операции - отложенное выполнение вычислений
REST API - HTTP-взаимодействие между сервисами
Short Polling - механизм real-time обновления данных
Мультиязычная архитектура - использование разных языков для сервисов
Токен-авторизация - упрощенная система безопасности API
