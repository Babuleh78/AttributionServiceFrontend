# Лабораторная работа 6

## Цели работы

- **Внедрение адаптивности** - обеспечение корректного отображения приложения на устройствах с разными размерами экранов
- **Реализация PWA (Progressive Web App)** - добавление возможности установки приложения и работы в оффлайн-режиме
- **Создание нативного приложения** с использованием Tauri framework
- **Развертывание приложения** на GitHub Pages
- **Внедрение менеджера состояний** Redux Toolkit для управления состоянием фильтров

## Основные задачи

- Добавление адаптивности для трех страниц приложения
- Настройка PWA-функциональности
- Создание Tauri-приложения для гостевого интерфейса
- Подключение к API через IP-адрес в локальной сети
- Развертывание фронтенда на GitHub Pages

## Технологии

- Redux Toolkit (менеджер состояний)
- PWA (Progressive Web App)
- Tauri (нативное приложение)
- GitHub Pages (деплой)
- Адаптивный дизайн
